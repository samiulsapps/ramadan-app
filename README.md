## ğŸš€ Getting Started

### Prerequisites
- Git installed on your machine
- A text editor (VS Code, Sublime Text, etc.)
- Basic knowledge of HTML/CSS/JavaScript

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/samiulsapps/ramadan-app.git
   cd ramadan-app
   ```

2. **Open in browser**
   - Simply open `index.html` in your browser
   - Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. **Visit in browser**
   ```
   http://localhost:8000
   ```
